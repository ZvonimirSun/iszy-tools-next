一个基于 Nuxt 4 的前端工具集应用。

## 项目安装

```shell
pnpm install
```

### 启动开发服务器

```shell
pnpm dev
```

### 为生产环境构建产物

```shell
pnpm build
```

### 本地预览生产构建产物

```shell
pnpm preview
```
